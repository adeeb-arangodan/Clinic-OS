# Architecture Decision Records

Decisions not already covered by `docs/00`–`04` are recorded here, one file per
decision, numbered sequentially: `NNNN-short-title.md`.

Template:

```markdown
# NNNN — Title

- **Status:** proposed | accepted | superseded by NNNN
- **Date:** YYYY-MM-DD
- **Context:** what forced the decision
- **Decision:** what we chose
- **Consequences:** trade-offs accepted
```

The six foundational ADRs are summarized in `docs/02-ARCHITECTURE.md` §10.
